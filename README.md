# 🌱 TerraCult — Smart Agriculture Desktop Application

> Java desktop application for connected and intelligent agriculture management.

---

## 📌 Project Overview

TerraCult is a multi-module desktop application designed to modernize agricultural 
management. It provides tools for managing agricultural equipment, products, and 
resources through an intuitive JavaFX interface connected to a MySQL database.

---

## 🧩 My Contribution — Agricultural Equipment Management Module

- Built a full CRUD interface for managing agricultural materials and equipment
- Developed the product listing page with filtering and sorting features
- Implemented detailed product view with images and technical descriptions
- Integrated an AI API for automatic generation of technical descriptions
- Connected the JavaFX interface to MySQL database using JDBC

---

## 🛠️ Tech Stack

- **Language :** Java
- **UI Framework :** JavaFX
- **Database :** MySQL (via JDBC)
- **AI Integration :** AI API for automatic description generation
- **Tools :** Scene Builder, IntelliJ IDEA
